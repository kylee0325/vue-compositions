vue-compositions #

vue-compositions 是基于 Composition API 开发的组合函数库。基于vue-demi库搭建，可以同时兼容 vue2 和 vue3。

Inspiration #

这个项目的灵感来源于一些优秀的开源 React Hooks，如： react-use、ahooks等。

Install #

npm install vue-compositions
yarn add vue-compositions
